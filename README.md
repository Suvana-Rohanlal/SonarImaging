## Usage
	Simulation:
	1. Run simulate.jl
	2. Run signalProcessing.jl
	3. Run image.jl 
  
	Prototype:
	1. Install arduino and teensyduino
	2. Run any of the implementations.
	3. Edit and Run plotImage.jl to compute the serial communication, signal processing and image formation.
